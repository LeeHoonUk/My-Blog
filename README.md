## 2. Coding Convention

> 여러사람이 협업을 해도 모두가 읽기 편한 코드를 작성하기 위한 기본 규칙   
> But Local Rule이 더 중요!!

```
 * 한 줄의 문자열은 79자, Django는 119자 추천
 * DocString은 72자
 * snake_case 사용
 * 모듈 레벨 상수는 모두 대문자
 * ClassName은 Capitalized Word
 * 한 줄로 된, if, try...except, for, while 구문은 사용하지 않는다.
```
